“Brain Games” is a set of five console math games for brain training.

## Setup

```bash
git clone https://github.com/tiltovskee/frontend-project-44.git

cd frontend-project-44

make install
```

## System requirements

* Ubuntu 22.04.3 or higher
* Node.js 20.7.0 or higher
